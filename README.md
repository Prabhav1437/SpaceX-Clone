# SpaceX Clone

A premium, high-performance web application cloning the SpaceX digital experience. This project features immersive vehicle pages, smooth motion effects, and a fully functional shop with real-time cart management.

## 🚀 Features

### Immersive Vehicle Experiences
- **Dynamic Pages**: Dedicated, full-screen experiences for Starship, Dragon, Falcon 9, and Falcon Heavy.
- **Human Spaceflight**: Detailed overview of missions to the ISS, Moon, Mars, and Earth Orbit.
- **Cinematic Visuals**: High-resolution imagery and video backgrounds tailored for a premium feel.

### Integrated Shop
- **Product Discovery**: Storefront with filtering by category (Shirts, Outerwear, Accessories).
- **Interactive Previews**: Image-swap on hover to see front and back views of merchandise.
- **Basic Cart System**: 
    - Real-time cart updates without complex state libraries.
    - Smooth sliding cart overlay for managing quantities.
    - Persistent counts across the shopping experience.

### Technical Excellence
- **Smooth Scrolling**: Integrated **Lenis** for a silky, modern scrolling feel.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop using **Tailwind CSS**.
- **Performance**: Built on **Vite** for lightning-fast development and optimized production builds.

## 🛠 Tech Stack

- **Frontend**: React.js
- **Build Tool**: Vite
- **Styling**: Tailwind CSS (v4)
- **Motion**: Lenis (Smooth Scroll)
- **Routing**: React Router DOM

## 📦 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Prabhav1437/SpaceX-Clone.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

## 📂 Project Structure

- `src/components`: Reusable UI elements (Navbar, Footer, Hero, etc.)
- `src/pages`: Main view components for vehicles, spaceflight, and shop.
- `src/data`: JSON product catalog and mission data.
- `src/context`: Simplified state management (where applicable).
